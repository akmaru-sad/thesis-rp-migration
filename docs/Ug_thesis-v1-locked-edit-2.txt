                   mismatch concern (BD-local BMD insufficient for
                   ~536,000 km² catchment dominated by upstream
                   Tibet/Assam/Bhutan precipitation). Methodological
                   limitation persists at the routing level; document
                   explicitly per R6+R8.
   D.2  Multiple Grubbs-Beck low-outlier removal (USGS PeakFQ)
   D.3  Multi-distribution FFA on historical AMS (5 candidates):
        GEV, Gumbel (EV1), LN3, LP3, GPD-POT
        Selection: AIC + BIC + Anderson-Darling combined ranking
        Tools    : R (lmom + lmomRFA + fitdistrplus); Python
                   (scipy.stats + lmoments3) cross-check
   D.4  Same FFA on future AMS per (GCM, SSP)
   D.5  Q_T tabulation: T ∈ {2, 5, 10, 25, 50, 100} per scenario
   D.6  RP-MIGRATION computation:
        For each (GCM, SSP, T_hist ∈ {25, 50, 100}):
          Q_T_hist = F_hist^{-1}(1 − 1/T_hist)
            where F_hist = non-exceedance CDF of historical
            best-fit distribution
          p_nonexceed_fut = F_fut(Q_T_hist)
            where F_fut = non-exceedance CDF of future best-fit
            distribution per (GCM, SSP)
          p_exceed_fut   = 1 − p_nonexceed_fut
          T_future       = 1 / p_exceed_fut
        Boundary handling (in main algorithm):
          if T_future > 500: report "T_future > 500 yr (saturated)"
          if T_future < 1:   report "T_future < 1 yr (annual)"
   D.7  EFR = T_hist / T_future per (station, GCM, SSP)
   D.8  Ensemble percentiles: report min / median / max across
        3 GCMs
